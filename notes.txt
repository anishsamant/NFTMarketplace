1) configure the truffle.config file to link ganache and truffle.
2) create and initialize the contract deployment js.


$ truffle init                      // to create a truffle project
$ truffle compile                   // to compile the contract
$ truffle migrate                   // to deploy
$ truffle console                   // to open truffle console to test
$ truffle migrate --reset           // to reset our migration
$ truffle test                      // to execute tests

ERC165 interface is used to check whether our contract is compliant
